# engr-course-templates
Reusable Latex templates and workflows for undergraduate engineering course

This repository contains reusable Latex templates and structure workflows for creating accreditation-ready lesson plans, lab, quizzes, and assessments for undergraduate engineering courses.

2. Who is it for?


Designed for engineering lecturers, instructors, and curriculum designers who want consistent, professional course materials.

3. What’s inside?

Bullet list of folders (even if some are empty initially).

	•	Lesson plan templates
	•	Quiz and MCQ generators
	•	Teacher / student mode LaTeX switches
	•	Example course structures

4. How to use it (minimal)

One concrete step.

Clone the repository and compile the example LaTeX file in Overleaf or locally.
